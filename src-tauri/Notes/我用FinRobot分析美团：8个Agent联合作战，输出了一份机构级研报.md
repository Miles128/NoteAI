# 我用FinRobot分析美团：8个Agent联合作战，输出了一份机构级研报

## 背景：史上最惨年报配上暴涨阳线

2026年3月26日，美团发布2025年全年业绩。

营收3649亿元，同比增长8%。净亏损234亿元，由盈转亏。核心本地商业从2024年贡献524亿利润，到2025年亏损69亿。

一年蒸发近600亿利润。

然后，监管喊话了。3月25日《经济日报》发文《外卖大战该结束了》，市场监管总局官网转发。美团股价当天暴涨14%。

涨的不是业绩，是预期。

我看着这份"史上最惨年报"配上这根阳线，又想到了同一个问题：让AI帮我分析一遍，它会得出什么结论？

这次换个工具。**今天用的工具：FinRobot**

上一篇用TradingAgents-CN做泡泡玛特，更像一个会吵架的分析师团队，最后投票得出结论。FinRobot不一样。它的目标是输出一份机构级研报。

## FinRobot 的架构

FinRobot用三层Agent协作：

- **Data-CoT Agent**：负责数据收集，从财报、新闻、港交所披露等渠道抓取原始信息
- **Concept-CoT Agent**：负责投资逻辑提炼
- **Thesis-CoT Agent**：负责最终综合，生成完整的研究报告，包括估值、风险、情景分析

整个流程更像投行研究部的工作方式：先收集，再消化，最后成文。

## 第一步：Data Agent 收集了什么？

Data Agent跑完后，吐出了一堆原始素材：

### 财务数据

- 销售及营销开支从640亿元飙升至1029亿元，增幅超60%

### 竞争格局数据

- 订单量份额：美团约50%，阿里约42%，京东约8%
- 美团每单亏损约1.2-1.4元，阿里每单亏损约3.0元

### 海外业务进展

- Keeta已进入香港、沙特、卡塔尔、科威特、阿联酋、巴西

### 其他关键数据

- Q4核心本地商业经营亏损100亿元，较Q3环比大幅收窄

## 第二步：Concept Agent 提炼了什么投资逻辑？

Concept Agent把上面的数据，转化成了几条核心投资逻辑：

### 逻辑一：防御性亏损，不是模式崩塌

美团2025年的巨额亏损，本质是"用最小的代价守住最大的份额"。

数据支撑：美团用234亿净亏损，守住了60%以上的GTV份额。阿里投入更多，份额提升到42%，但每单亏损是美团的2倍以上。

结论：这是战术亏损，不是战略失败。

### 逻辑二：监管拐点已现

市场监管总局转发《外卖大战该结束了》，信号明确。2025年二三季度的疯狂补贴大概率不会重演。

数据支撑：美团股价当天暴涨14%，市场已经在为"补贴战结束"定价。

结论：最差的时刻可能已经过去。

### 逻辑三：第二曲线尚在验证

海外业务Keeta进展快于预期，但仍在投入期。AI投入持续加码，但商业化路径不清晰。

数据支撑：新业务2025年亏损超100亿元，其中海外前期投入是主因。

结论：第二增长曲线存在，但还需要时间验证。

## 第三步：Thesis Agent 写出了什么结论？

Thesis Agent综合以上分析，给出了一份完整的投资判断：

### 估值分析

- 当前股价约80港元，对应2026年预测市盈率约16倍
- 机构目标价区间：104-200港元，中位数约140港元

### 核心风险

- 补贴战反复：如果监管信号未能有效落地，价格战可能重燃
- 份额流失：即便补贴减弱，用户习惯已经被竞争对手部分改变

### 情景分析

- 乐观情景（监管有效+份额稳定+新业务减亏）：目标价180港元+
- 中性情景（竞争缓和但利润修复慢）：目标价120-140港元
- 悲观情景（补贴战延续+份额继续下滑）：目标价80港元以下

### 最终信号

**持有观望，不加仓，不减仓。**

理由：估值已到历史低位，监管拐点已现，但新增长引擎尚未出现，上行催化剂不明确。在确认性信号出现之前，保持中性。

## 我的补充判断

部分同意AI的结论，但有一点我想补充。

AI分析的框架是对的：估值低，但增长逻辑未验证。这是一个典型的"便宜但不一定值得买"的状态。

但有一个维度是AI没有充分权重的：**这是一场"谁先撑不住"的游戏。**

外卖补贴战的本质，不是谁能打赢，而是谁能撑到最后。

阿里的闪购业务还在亏钱，而且亏得比美团多。京东外卖的市场份额只有8%左右，还在烧钱获客阶段。

美团的优势是什么？

- **每单亏损最低**
- **配送网络最密**
- **用户心智最强**

当监管喊停补贴战，谁的利润恢复最快？答案显然是美团。

这也是为什么监管一喊话，美团能涨14%——市场知道，美团是补贴战结束的最大受益者。

所以我的判断比AI更乐观一点：如果补贴战确实在收敛，美团的利润修复速度会比竞争对手快得多。60-70港元的区间，是我认为比较有性价比的建仓位置。

## FinRobot vs TradingAgents：两种AI投研风格

跑完两个工具，我发现它们的定位完全不同：

- 如果你**对一家公司完全陌生**，FinRobot的研报式输出更适合建立全局认知
- 如果你**已经有了初步判断**，TradingAgents的多空辩论更能帮你发现盲点

两个工具不是替代关系，是互补关系。

## 一个关于AI研报的真实体会

跑完FinRobot，我发现了一个有意思的现象：

AI给出的结论是"持有观望"，但市场在3月25日给出的结论是"暴涨14%"。

谁对？

其实都对。AI是在分析基本面，市场是在交易预期。AI看到的是"利润还没恢复"，市场看到的是"最坏的时候过去了"。

这让我意识到，FinRobot这类工具的真正价值，不在于它给你一个买卖信号。**而在于它把"你需要判断什么"摆到了台面上。**

比如这次，FinRobot帮我梳理清楚了：美团的核心变量就两个——补贴战什么时候结束，以及结束后份额能守住多少。

没有用AI之前，我可能会被"净亏损234亿"这个数字吓到，也可能会被"研发投入260亿"这个故事带跑。

有了AI，这些噪音被过滤掉了，真正的问题被逼出来了。

至于这些问题的答案，AI给不了你。那是你自己要下的注。
