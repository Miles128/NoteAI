NoteAI
v1.0.0

NoteAI 是一款智能知识管理工具，帮助用户高效组织和管理 Markdown 笔记。

功能：

  • 多层主题树知识图谱
  • AI 辅助写作与内容整合
  • 网页内容下载与分析
  • 标签与双向链接管理

技术栈：Tauri v2 + Python Sidecar + D3.js

基于 Rust 构建桌面端，Python 提供 AI 能力，D3.js 渲染知识图谱。
