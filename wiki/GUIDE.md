# MyNotes 工作区指南

本文件定义 MyNotes 工作区的内容组织规范。

---

## 一、三级主题体系

### 一级主题（4 个领域）

| 一级主题 | 收录内容 |
|---------|---------|
| **普通人的AI指南** | AI 科普入门；Agent 是什么、市面上有哪些 Agent 怎么用（面向普通用户） |
| **AI应用开发教程** | 硬核开发：RAG 架构、模型微调、各种 Agent 怎么开发、编程工具用法 |
| **AI产品经理之路** | 新产品评测、行业动态、产品分析 |
| **AI使用技巧和信息** | Skills 用法、技巧分享；特定 Agent 平台教程（Hermes Agent、OpenClaw、Cloud Code 等具体产品） |

### 层级规则

- 一级 = 领域（以上 4 个，不可超出）
- 二级 = 方向（如 `AI应用开发教程 > RAG 知识库架构`）
- 三级 = 子题（如 `AI应用开发教程 > RAG 知识库架构 > Milvus 向量数据库`）
- 最多三层，三级下不再设子题

### 主题存储格式

- YAML frontmatter: `topic: 一级 > 二级 > 三级`
- 文件系统: `Notes/一级/二级/三级/文件名.md`
- 分隔符: ` > `（空格 + 大于号 + 空格）

---

## 二、目录结构

```
MyNotes/
├── Notes/                    # 笔记（按主题分文件夹）
│   ├── 普通人的AI指南/
│   ├── AI应用开发教程/
│   ├── AI产品经理之路/
│   └── AI使用技巧和信息/
├── wiki/                     # AI 编译知识
│   ├── WIKI.md               # 全局主题索引
│   └── GUIDE.md              # 本文件
├── Raw/                      # 非 Markdown 文件归档（PDF、DOCX、PPTX、图片等）
└── .ai_memory/               # AI 记忆数据
```

- **Notes/**: 所有 Markdown 笔记。按主题层级分文件夹。
- **wiki/**: AI 生成的结构化产物。
- **Raw/**: 原始格式文件。PDF、Word、PPT、图片等非 MD 文件统一放这里。
- 其他临时文件、导出文件等放在对应主题文件夹下。
